Time-series analysis of strategic food commodity prices and inflation volatility in West Java and Jakarta Indonesia (2022-2025) using Python (Pandas, Seaborn) ETL Pipeline.

# Food Price Volatility Analysis (Jakarta & West Java)
This project analyzes food price fluctuations to detect inflationary anomalies and long-term trends using Python. It serves as an early warning system prototype for data-driven policy intervention.

## 🚀 Key Features
- **ETL Pipeline:** Automated cleaning and merging of multiple raw CSV datasets using Pandas.
- **Time-Series Analysis:** Applied Moving Average (MA) smoothing to visualize price trends vs. actual volatility.
- **Anomaly Detection:** Implemented absolute growth logic to identify extreme inflation/deflation.

## 🛠 Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook

## 📝 Data Source
- Source: https://www.kaggle.com/datasets/muhyusuf1112/indonesia-commodity-price-based-piphs-source
- Coverage: Daily retail prices for 10 strategic commodities across 34 Indonesian provinces.
- Timeframe: January 1, 2022 – February 12, 2026 (~1,460 days).
- Scale: 500,000+ data points surveyed daily from 82 traditional markets nationwide.
- Primary Source: bi.go.id/hargapangan
